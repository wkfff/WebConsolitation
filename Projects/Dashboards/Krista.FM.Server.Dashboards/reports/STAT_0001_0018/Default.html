Назначение:отчет предназначен для анализа динамики и структуры инвестиций в основной капитал в субъекте РФ в разрезе:
  • видов деятельности;
  • форм собственности;
  • источников финансирования;
  • видов основных фондов.
   Параметры: выбирается:
  • территория: муниципальные образования, входящие в субъект РФ, либо субъект РФ в целом;
  • разрезность: по видам экономической деятельности, по формам собственности, по источникам финансирования, по видам основных фондов.
   После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация в целом по субъекту РФ в разрезе видов экономической деятельности.
   Для выбранного муниципального образования (субъекта РФ) формируется:
  • структурная диаграмма по показателю «Структура инвестиций в основной капитал» за последний квартал, по которому есть данные в системе;
  • диаграмма динамики и структуры по показателю «Структура инвестиций в основной капитал».
   В таблице наглядно представлена структура и динамика инвестиций в основной капитал, а также темп роста к аналогичному периоду предыдущего года.
