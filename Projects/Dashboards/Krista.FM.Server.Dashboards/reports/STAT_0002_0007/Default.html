Назначение: отчет предназначен для анализа вклада по численности безработных граждан в ситуацию в УрФО и в РФ субъектов Российской Федерации, входящих в Уральский федеральный округ.

Параметры: выбирается показатель: «Общая численность безработных (по методологии МОТ)» или «Число безработных зарегистрированных в службе занятости».

По показателю «Число безработных зарегистрированных в службе занятости» данные выводятся на последнюю дату месяца за которую есть данные, в текущем месяце берутся данные на последнюю даты за которую есть данные.

После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация по показателю « Общая численность безработных (по методологии МОТ)» за последний период, по которому есть данные. Для выбранного показателя формируются круговые диаграммы соотношения вклада субъектов в Уральский федеральный округ и вклада Уральского федерального округа в РФ.
