Назначение:отчет предназначен для сравнительного анализа территорий по основным показателям востребованности трудовых ресурсов.
   Параметры:выбирается период (дата, на которую проводится анализ данных). После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю отчетную дату, по которой имеются данные.
   В отчёте отображается карта рассматриваемой территории с условной раскраской в соответствии со значениями выбранного для анализа показателя. Анализ проводится по следующим показателям:
   Анализ проводится по следующим показателям
    • «Численность занятых индивидуальным трудом и по найму у отдельных граждан, в % к численности трудовых ресурсов»
    • «Численность граждан, занятых в домашнем хозяйстве производством товаров и услуг для реализации, в % к численности трудовых ресурсов»
    • «Трудоспособное население в трудоспособном возрасте, не занятое в экономике и обучением, в % к численности трудовых ресурсов»
    • «Среднесписочная численность работников организаций и предприятий, в % к численности трудовых ресурсов»
   По выбранному показателю определяются ранги территорий и выставляются звёзды-маркеры: «золотая звезда» - у территории с наилучшим значением показателя, «серая звезда» - у территории с наихудшим значением показателя.
   По тому же показателю, что отображается на карте, дополнительно выводится гистограмма, отражающая значение показателя по всем территориям в порядке возрастания значений, в сравнении с уровнем по субъекту РФ.
