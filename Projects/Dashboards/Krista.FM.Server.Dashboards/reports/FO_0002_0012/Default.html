Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, бюджета субъекта РФ и местных бюджетов.
Назначение:Отчет предназначен для оценки исполнения поступлений доходов в местные бюджеты, а также для оценки структуры поступлений доходов местных бюджетов в общей сумме консолидированного бюджета муниципальных образований.
Параметры: выбираются год, месяц и вид доходов. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний год и месяц, по которым есть данные в базе, и по виду доходов «Доходы ВСЕГО».
В таблице по уровням бюджета приводятся следующие показатели:
  • утвержденные годовые назначения;
  • исполнение;
  • % исполнения годовых назначений;
  • удельный вес поступлений бюджета района (поселения) в консолидированном бюджете МО;
  • ранг по удельному весу поступлений бюджета района (поселения) в консолидированном бюджете МО. Желтой звездой отмечено муниципальное образование с самым большим удельным весом, серой – с самым маленьким.
