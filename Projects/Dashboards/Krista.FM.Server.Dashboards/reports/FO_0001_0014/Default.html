Источник данных:отчетность об исполнении бюджетов, база планирования бюджета.
   Назначение:отчет предназначен для анализа структуры доходов местных бюджетов.
   Параметры:
Выбирается одна из опций:
a. «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
b. «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
c. «Проект бюджета» - в отчет выводятся данные проектов бюджетов по доходам.
Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» «Месяц» (по умолчанию установлены последние, по которым имеются данные), «Бюджет» (можно выбрать все местные бюджеты, один из муниципальных районов или городских округов). При выборе муниципального района появляются дополнительные параметры «Консолидированный бюджет МР», «Бюджет муниципального района», «Свод бюджетов поселений». По умолчанию в отчете отображаются данные по консолидированному бюджету района. Чтобы посмотреть данные по собственному бюджету муниципального района, бюджетам поселений необходимо в параметрах выбрать соответственно «Бюджет муниципального района», «Свод бюджетов поселений».
После изменения параметров нажмите кнопку «Обновить».
В таблице отображаются:
- Сумма фактического исполнения (бюджетных назначений, проектные данные) за аналогичный период предыдущего года.
- Сумма фактического исполнения (бюджетных назначений, проектные данные) по соответствующим доходным источникам.
- Доля данного вида доходного источника (группы доходных источников) в общей сумме доходов бюджета.
- Темп роста доходов к аналогичному периоду прошлого года
По выбранному периоду и бюджету отображается круговая диаграмма с частью значений, вынесенных в гистограмму (межбюджетные трансферты).
