Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, собственного бюджета субъекта РФ и местных бюджетов.
Назначение:отчет предназначен для анализа изменений плановых и фактических данных по налоговым доходам бюджетов субъекта РФ
Параметры: выбирается год, месяц, бюджет. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний отчетный период по консолидированному бюджету субъекта.
Для анализа уточненных годовых назначений по налоговым доходам необходимо включить параметр «План», а для анализа исполнения – «Факт». В таблице отчета приводятся данные по следующим показателям:
  • сумма уточненных годовых назначений помесячно за выбранный в параметре год или фактическое исполнение по месяцам выбранного в параметре года;
  • отклонение от прошлого месяца;
  • отклонение выбранного в параметре месяца от января текущего года.
 Диаграмма позволяет сравнить помесячное поступление налоговых доходов в разбивке по основным группам налоговых доходов или сравнить изменение годовых назначений бюджета по группам налоговых доходов помесячно за анализируемый период.
