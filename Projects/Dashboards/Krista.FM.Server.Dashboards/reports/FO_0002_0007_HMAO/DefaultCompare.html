Назначение:отчет предназначен для сравнительного анализа исполнения по расходам и бюджетной обеспеченности муниципальных образований субъекта РФ.
   Параметры:выбираются год и месяц (по умолчанию установлены последние, по которым имеются данные), раздел (подраздел) классификации расходов. После изменения параметров нажмите кнопку «Обновить».
   В таблице по каждому муниципальному образованию приводятся:
  • фактическое исполнение по выбранному разделу (подразделу) расходов;
  • плановые годовые назначения;
  • процент исполнения плана с индикатором сравнения со средним исполнением по региону (красный индикатор – ниже среднего, зеленый индикатор - выше среднего);
  • численность постоянного населения;
  • бюджетные расходы на душу населения по выбранному разделу (подразделу) классификации расходов;
  • ранг по Бюджетным расходам на душу населенияв субъекте РФ (лучшее муниципальное образование отмечается золотой звездой, худший – серой);
  • доля расходов данного вида в общем объеме расходов бюджета;
  • ранг местного бюджета по доле расходов данного вида в общем объеме расходов бюджета.
Для бюджетов муниципальных районов на карте заливкой отображаются бюджетные расходы на душу населения по выбранному разделу (подразделу) классификации расходов
