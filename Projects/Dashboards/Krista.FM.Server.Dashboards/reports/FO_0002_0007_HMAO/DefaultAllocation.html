Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, бюджета субъекта РФ и местных бюджетов.
   Назначение:отчет предназначен для сравнительного анализа бюджетных расходов на душу населения муниципальных образований субъекта РФ.
   Параметры:выбираются год, месяц (по умолчанию установлены последние, по которым имеются данные), территория, раздел (подраздел) классификации расходов. После изменения параметров нажмите кнопку «Обновить».
   В отчете отображается точечная диаграмма. Каждое муниципальное образование представлено точкой. По оси X расположена численность населения. Чем дальше точка вправо, тем выше численность населения в муниципальном образовании. По оси Y расположены бюджетные расходы на душу населения. Чем выше точка, тем выше бюджетная обеспеченность. Пунктирной линией приведено среднее значение бюджетных расходов на душу населения по субъекту РФ.
   Точка выбранного в параметрах муниципального образования подсвечивается оранжевым цветом. При наведении курсора мыши на точку выводится подсказка с информацией по муниципальному образованию.
   Отчет позволяет визуально анализировать распределение территорий по бюджетным расходам на душу населения.
