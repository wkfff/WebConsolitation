Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, собственного бюджета субъекта РФ и местных бюджетов, данные по численности населения субъекта.
   Назначение:отчет предназначен для сравнительного анализа среднедушевых доходов муниципальных образований субъекта РФ.
  Параметры:выбираются год, месяц (по умолчанию установлены последние, по которым имеются данные) и территория. После изменения параметров нажмите кнопку «Обновить».
   В отчете отображается точечная диаграмма. Каждое муниципальное образование представлено точкой. По оси X расположена численность населения. Чем дальше точка вправо, тем тем выше численность населения в муниципальном образовании. По оси Y расположены среднедушевые доходы. Чем выше точка, тем больше показатель среднедушевых доходов. Пунктирной линией приведено среднее значение среднедушевых доходов.
   Точка выбранного в параметрах муниципального образования подсвечивается оранжевым цветом. При наведении курсора мыши на точку выводится подсказка с информацией по муниципальному образованию.
   Отчет позволяет визуально анализировать распределение территорий по среднедушевым доходам.
   Если установить отметку в поле «Не учитывать дотации на выравнивание бюджетной обеспеченности», то из объема доходов будут исключены эти дотации.
