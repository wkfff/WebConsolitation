Назначение: отчет предназначен для сравнительного анализа расходов на содержание и численности работников органов местного самоуправления в разрезе главных распорядителей средств бюджета.

Параметры: выбирается период, который необходимо проанализировать. После изменения параметров нажмите кнопку «Обновить».

В таблице по каждому главному распорядителю бюджетных средств отображаются:

  • количество подведомственных учреждений;
  • количество государственных гражданских служащих;
  • количество работников;
  • расходы на содержание;
  • начисления на выплаты по оплате труда.
