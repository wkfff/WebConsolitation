Назначение: отчет представляет собой отображение показателей эффективности деятельности органов местного самоуправления в разрезе муниципальных округов.
Параметры: выбирается значение параметра «Показатель». В выпадающем списке выводятся все показатели, необходимые для проведения оценки. Показатели оценки эффективности при этом в выпадающем списке помечаются звездочкой. После выбора показателя нужно нажать кнопку «Обновить». По умолчанию, в параметре автоматически установлен первый показатель из списка.
    Выбирается значение параметра «Состояние данных». По умолчанию в параметре выбрано «Все состояния».
    В отчете представлена таблица, содержащая значения выбранного показателя в разрезе муниципальных образований. В таблицу выводятся данные за завершенный период, отчетный период с указанием плановых значений на три последующих года, а также примечание. По значениям показателя за отчетный период в отчете рассчитывается темп роста и место показателя.
    В отчете представлена гистограмма по всем МО по выбранному показателю в порядке убывания показателя. Для показателей, по которым имеются среднеобластные значения показателей или нормативы, они отражаются на диаграмме линией уровня.
