Источник данных: данные отчета об исполнении консолидированного бюджета субъекта (на основании приказа Минфина РФ №128н).
   Назначение: отчет предназначен для оценки исполнения бюджетов в разрезе отдельных показателей.
   Параметры:    выбирается год и месяц. После изменения параметров нажмите кнопку «Обновить». По умолчанию в отчете выводится информация за последний период, по которому есть данные.
   В таблице отчета приводятся следующие показатели:
  • общая сумма доходов, в том числе налоговые и неналоговые доходы, безвозмездные поступления;
  • общая сумма расходов, в том числе оплата труда, капитальные вложения и материальные затраты;
  • дефицит \ профицит бюджета.
   По каждому из этих показателей в отчете приводится:
  • утвержденные годовые назначения;
  • фактическое исполнение;
  • процент выполнения годовых назначений с цветовой оценкой равномерности исполнения (1/12 годового плана в месяц).
