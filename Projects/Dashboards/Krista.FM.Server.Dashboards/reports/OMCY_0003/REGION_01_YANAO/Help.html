Назначение: отчет предназначен для анализа и оценки уровня эффективности деятельности органов местного самоуправления на основании оценки результатов деятельности, удовлетворенности населения и эффективности расходования бюджетных средств в каждой из сфер по выбранному муниципальному образованию.
Параметры:  в отчете применяются параметры «Период» и «Территория». Для таблицы показателей используется параметр «Детализация».
Если галочка в параметре «Детализация» не стоит (по умолчанию), то в таблицу выводятся показатели Уровней эффективности по всем сферам и рассчитанный на основании их показатель "Оценка эффективности деятельности ОМСУ".
Кроме того, можно просмотреть значения исходных показателей, на основании которых осуществляется оценка, а также расчетных промежуточных показателей оценки результативности деятельности ОМСУ, эффективности расходования бюджетных средств, оценки населением результатов деятельности ОМСУ по каждой из оцениваемых сфер. Для этого нужно поставить галочку в параметре «Детализация». Рассчитанные коэффициенты представлены в виде дробного числового значения от 0,00 до 1,00, характеризующего отклонение значения по данному муниципальному образованию от наилучшего значения среди всех муниципальных образований. Чем больше значение коэффициента, тем предпочтительнее положение данного муниципального образования среди муниципальных образований, принимающих участие в ранжировании. По значениям показателей для муниципального образования выставляются ранги (места). Для визуального отображения в ячейке с первым местом ставится желтая звездочка и серая звездочка у показателя, занявшего последнее место. В отчете по выбранному муниципальному образованию строится диаграмма с показателями Уровней эффективности по всем сферам и итоговому показателю "Оценка эффективности деятельности ОМСУ".
