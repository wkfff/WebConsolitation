Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, собственного бюджета субъекта РФ и местных бюджетов, статистические данные Самарастат.
Назначение:отчет предназначен для сравнения бюджетной обеспеченности доходами бюджетов городских округов и муниципальных районов.
Параметры:выбираются год (по умолчанию установлен текущий) и месяц (по умолчанию последний месяц за который есть данные). После изменения параметров нажмите кнопку «Обновить».
В таблице по каждому муниципальному образованию отображается:
  • плановые и фактические показатели по бюджетной обеспеченности за текущий и отчетный год;
  • фактические показатели по бюджетной обеспеченности на текущий месяц текущего года.
