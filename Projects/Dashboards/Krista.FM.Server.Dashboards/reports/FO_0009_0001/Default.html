Источник данных:результаты расчетов отдела межбюджетных отношений Департамента финансов Ярославской области по распределению дотации на выравнивание бюджетной обеспеченности муниципальных районов (городских округов).
Назначение:отчет предназначен для получения информации о распределении дотаций на выравнивание бюджетной обеспеченности муниципальных районов (городских округов) Ярославской области.
Параметры: выбирается период и вариант. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю дату, по которой имеются данные.
