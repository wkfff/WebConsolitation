Источник данных: АС Бюджет.

Назначение: Отчет предназначен для анализа исполнения расходов областного бюджета за анализируемый период в разрезе кодов бюджетной классификации.

Параметры: выбираются год и комбинация КБК. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний период, за который имеются данные.

В таблице отчета приводятся следующие показатели:

  • утвержденный годовой план по расходам по сводной бюджетной росписи;
  • исполнение по расходам нарастающим итогом с начала года.

Индикация в таблице отражает равномерность исполнения плана по расходам по имеющимся данным на момент формирования отчета.
