Источник данных:оперативная информация АС "Бюджет" об исполнении кассового плана бюджета автономного округа.
Назначение:предназначен для оперативного мониторинга исполнения кассового плана бюджета автономного округа.
Параметры:день. После изменения параметра нажмите кнопку «Обновить». По умолчанию данные выводятся за последний день, за который имеются сведения.
В отчете по каждому показателю выводятся годовой кассовый план и кассовое исполнение нарастающим итогом с начала года на выбранную дату, а также процент исполнения плана.
Дополнительно отчет позволяет оценить отклонение процента исполнения кассового плана от среднего значения по всем ГРБС по каждому виду расходов. Данные по расходам приводятся в разрезе следующих показателей:
  • заработная плата (КОСГУ 210);
  • капитальные вложение (КОСГУ 310);
  • коммунальные услуги (КОСГУ 223);
  • безвоздмездные перечисления другим бюджета бюджетной системы РФ (КОСГУ 251).
