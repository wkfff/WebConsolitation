Источник данных:ежедневная информация об исполнении кассового плана областного бюджета (Департамент финансов Ярославской области).
Назначение:отчет предназначен для анализа динамики и структурной динамики исполнения кассового плана областного бюджета.
Параметры:год, квартал и показатель кассового плана. После изменения параметра нажмите кнопку «Обновить». По умолчанию данные выводятся за текущий квартал.
В отчете по выбранному показателю кассового плана выводится на диаграмме динамика исполнения за квартал. Для составных показателей выводится динамика с отображением структуры значения показателя:

В таблице по выбранному в параметрах показателю выводятся плановые назначения, фактическое исполнение нарастающим итогом, процент исполнения квартального плана и фактическое исполнение за день.
Для остатка средств выводятся плановый остаток на конец квартала, фактическое значение остатка и изменение фактического значения остатка за период.
Для расходов в таблице дополнительно выводятся столбцы по отдельным видам расходов. Оценка равномерности расходования средств выводится цветовым индикатором. Оценка производится для каждого вида расходов по отдельному алгоритму:
  • Заработная плата: должны осуществляться выплаты до 6 и 21 числа каждого месяца не менее 1/6 части квартального плана;
  • Нормативные публичные обязательства: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Коммунальные расходы: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Межбюджетные трансферты: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Прочие расходы: равномерное расходование по числу дней квартала.
