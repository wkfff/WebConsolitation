Назначение:: отчет предназначен для сравнительного анализа поступления и недоимки по налогам и по виду экономической деятельности в разрезе муниципальных образований Самарской области.
   Параметры:выбирается год, который необходимо проанализировать, вид налога и вид экономической деятельности. После изменения параметров нажмите кнопку «Обновить».
В таблице по каждому муниципальному образованию отображаются:
• поквартальное поступление и недоимка по выбранному налогу и по виду экономической деятельности;
• темпы роста поступления налогов с учетом и без учета недоимки.
В диаграмме отображаются темпы роста поступления и недоимки по конкретному муниципальному образованию в помесячном разрезе. Для вывода информации в диаграмме необходимо выделить конкретное муниципальное образование в таблице отчета.
