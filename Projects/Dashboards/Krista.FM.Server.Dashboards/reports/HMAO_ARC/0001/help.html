Назначение:отчет предназначен для анализа структуры и динамики заболеваемости в субъекте РФ.
   Параметры: выбирается период. После выбора параметра необходимо нажать на кнопку «Обновить». По умолчанию в параметре автоматически устанавливается последний год, на который есть данные в системе.
   В таблице приведены значения показателя "Зарегистрировано больных с диагнозом, установленным впервые в жизни, на 1000 человек населения" по видам болезней, а также прирост, темп прироста, доля в общем объеме заболеваний за определенный период. Зеленая стрелка вверх иллюстрирует положительный темп прироста, красная стрелка вниз – отрицательный темп прироста. Есть возможность сортировки заболеваний по столбцам. При нажатии на строку с классом заболевания ниже на диаграмме строится динамика по выбранному показателю за последние несколько лет.
