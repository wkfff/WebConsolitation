Назначение: отчет предназначен для исследования и анализа ситуации по выбросам загрязняющих веществ в атмосферный воздух в субъекте РФ и его муниципальных образованиях
   Параметры: Параметры: Выбирается период. После выбора параметра необходимо нажать кнопку «Обновить». По умолчанию, в Параметре установлен последний период, на который есть данные.
   В таблице представлены показатели: «Всего выброшено в атмосферу загрязянющих веществ», «Разрешенный выброс в отчетном году», «Количество загрязняющих веществ, отходящих от всех стационарных источников выделения», «Выбрасывается без очистки загрязняющих веществ», «Выбрасывается без очистки загрязняющих веществ от организованных источников выбросов», «Поступает на очистные сооружения загрязняющих веществ», «Уловлено и обезврежено загрязняющих веществ», «Утилизировано загрязняющих веществ».
   Для всех показателей рассчитывается абсолютное отклонение и темп прироста относительно предыдущего периода. Для показателя «Всего выброшено в атмосферу загрязняющих веществ» рассчитывается отклонение и доля по сравнению с показателем «Разрешенный выброс в отчетном году». Для показателя «Уловлено и обезврежено загрязняющих веществ» рассчитывается доля по сравнению с показателем «Количество загрязняющих веществ, отходящих от всех стационарных источников выделения», для показателя «Утилизировано загрязняющих веществ» рассчитывается доля по сравнению с показателем «Уловлено и обезврежено загрязняющих веществ».
