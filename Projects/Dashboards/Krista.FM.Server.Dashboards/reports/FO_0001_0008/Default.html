Источник данных: отчетность об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, областного и местных бюджетов, база планирования бюджета.
   Назначение:отчет предназначен для сравнительного анализа основных показателей исполнения (планирования) консолидированного бюджета области, областного бюджета и местных бюджетов по доходам и расходам.
   Параметры:
Параметры:
1. Выбирается одна из опций:
a. «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
b. «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
c. «Проект бюджета» - в отчет выводятся данные проектов бюджетов по доходам и расходам.
2. Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» и «Месяц» (по умолчанию установлены последние, по которым имеются данные).
3. Для опции «Проект бюджета» в отчет выводятся параметры:
a. «Вариант доходов тек. год» - выбирается год, который является текущим, и вариант проекта доходов в указанном году;
b. «Вариант доходов пред. год» - выбирается год, с которым производится сравнение (базовый), и вариант проекта доходов в указанном году;
c. «Вариант расходов тек. год» - выбирается вариант расходов для текущего года (указанного в параметре «Вариант доходов тек. год»);
d. «Вариант расходов пред. год» - выбирается вариант расходов для базового года, с которым производится сравнение (указанного в параметре «Вариант доходов пред. год»).
4. После изменения параметров нажмите кнопку «Обновить».
В таблице по соответствующим уровням бюджетов отображаются:
· Сумма фактического исполнения (плановых назначений, проекта бюджета) по доходам и расходам.
· Сумма фактического исполнения (плановых назначений, проекта бюджета) по доходам и расходам за аналогичный период предыдущего года.
· Темп роста доходов и расходов к аналогичному периоду прошлого года (к базовому году для проекта бюджета).
· Сравнение темпов роста доходов и расходов.
По выбранным параметрам строятся две диаграммы в нижней части отчета.
Диаграмма слева позволяет сравнить фактические (плановые или проектные) данные по доходам в разрезе уровней бюджетов по выбранному периоду, диаграмма справа - фактические (плановые или проектные) данные по расходам.
При наведении курсора мыши на любую категорию диаграммы, автоматически выводятся комментарии.
