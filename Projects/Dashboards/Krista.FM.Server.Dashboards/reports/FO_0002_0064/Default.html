Назначение: отчет предназначен для анализа освоения бюджетных ассигнований, выделенных из бюджета субъекта на осуществление инвестиций в объекты капитального строительства государственной и муниципальной собственности.
Параметры:: В отчете выбираются следующие параметры: «Год», «Месяц» и «День». После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю дату, на которую имеются данные. В таблице отчета за выбранный период и аналогичный период предыдущего года отображаются: - годовой объем расходов бюджета инвестиционного характера; - кассовое исполнение расходов бюджета инвестиционного характера; - % кассового поступления к годовому объему расходов. Диаграммы отчета показывают сравнительную характеристику кассового исполнения расходов бюджета инвестиционного характера за выбранный период по сравнению с аналогичным периодом предыдущего года за счет средств: - областного бюджета; - федерального бюджета и иных безвозмездных поступлений (без учета средств, выделенных из федерального бюджета)
