Назначение:отчет предназначен для анализа исполнения консолидированного бюджета в разрезе отдельных показателей, а также показателей исполнения в помесячной динамике за несколько лет.
Параметры: выбираются год и месяц (по умолчанию выбирается последний, за который есть данные). После изменения параметров нажмите кнопку «Обновить».
В таблице отображается факт исполнения в разрезе отдельных показателей консолидированного бюджета.
Диаграмма отчета показывает динамику исполнения по выбранному показателю в помесячной динамике за несколько лет.
