Источник данных:ежедневная информация об исполнении кассового плана областного бюджета (Департамент финансов Ярославской области).
Назначение:отчет предназначен для оперативного мониторинга исполнения кассового плана областного бюджета.
Параметры:день. После изменения параметра нажмите кнопку «Обновить». По умолчанию данные выводятся за последний день, за который имеются сведения.
В отчете по каждому показателю выводятся квартальные назначения и фактическое исполнение на выбранную дату, а также процент исполнения квартального плана.
Дополнительно отчет позволяет оценить равномерность расходования средств. Оценка равномерности выводится цветовым индикатором в целом по расходам и для каждого ведомства. Оценка производится для каждого вида расходов по отдельному алгоритму:
  • Заработная плата: должны осуществляться выплаты до 6 и 21 числа каждого месяца не менее 1/6 части квартального плана;
  • Нормативные публичные обязательства: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Коммунальные расходы: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Межбюджетные трансферты: на конец месяца должно быть исполнено 1/3 квартального плана;
  • Прочие расходы: равномерное расходование по числу дней квартала.
