Назначение:отчет предназначен для сравнительного анализа исполнения по доходам и среднедушевых доходов муниципальных образований субъекта РФ.
Параметры:выбираются год и месяц (по умолчанию установлены последние, по которым имеются данные), вид дохода. После изменения параметров нажмите кнопку «Обновить».
В таблице по каждому субъекту приводятся:
  • фактическое исполнение по выбранному виду доходов;
  • плановые годовые назначения;
  • процент исполнения плана;
  • ранг по проценту исполнения в субъекте РФ (лучшее муниципальное образование отмечается золотой звездой, худшее – серой);
  • численность постоянного населения;
  • среднедушевые доходы по выбранному виду доходов;
  • ранг по среднедушевым доходам в субъекте РФ (лучшее муниципальное образование отмечается золотой звездой, худший – серой);
  • доля доходов каждого муниципального образования в доходах бюджетов МО;
  • доля доходов каждого муниципального образования в доходах консолидированного бюджета субъекта.

Для бюджетов муниципальных районов на карте заливкой отображается процент исполнения плановых назначений по выбранному доходному источнику.
