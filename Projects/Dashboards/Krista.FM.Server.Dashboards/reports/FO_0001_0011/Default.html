Источник данных:отчетность об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, областного бюджета и местных бюджетов, база планирования бюджета.
   Назначение:отчет предназначен для оценки удельного веса дотаций на выравнивание бюджетной обеспеченности муниципальных районов (городских округов) в объеме доходов местных бюджетов.
   Параметры:
Выбирается одна из опций:
a. «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
b. «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
«Проект бюджета» - в отчет выводятся данные проектов бюджетов.
Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» «Месяц» (по умолчанию установлены последние, по которым имеются данные), «Бюджет» (можно выбрать бюджет муниципального района или свод бюджетов поселений, входящих в состав муниципального района).
Для опции «Проект бюджета» в отчет выводится параметр «Вариант доходов», выбирается год, который является текущим, и вариант проекта доходов в указанном году
После изменения параметров нажмите кнопку «Обновить».
В таблице по соответствующему уровню бюджета отображаются:
- Сумма фактического исполнения (бюджетных назначений, проектные данные) дотаций на выравнивание бюджетной обеспеченности муниципальных районов (городских округов);
- Фактическое исполнение (бюджетные назначения, проектные данные) общей суммы доходов;
- Доля дотаций на выравнивание бюджетной обеспеченности муниципальных районов (городских округов) в доходах местных бюджетов.
По выбранным данным отображается линейчатая диаграмма.
