Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, бюджета субъекта РФ и местных бюджетов.
Назначение:отчет предназначен для анализа динамики структуры доходов.
Параметры:выбираются года, по которым проводится сравнение (по умолчанию три последних года), бюджет, а так же вид доходного источника. После изменения параметров нажмите кнопку «Обновить».
 При выборе параметра муниципальные районы или определенного муниципального района появляется дополнительный параметр «Консолидированный бюджет МО». По умолчанию этот параметр включен и в отчете отображаются данные с учетом консолидированного бюджета районов. Чтобы посмотреть данные с учетом бюджетов муниципальных районов, необходимо отключить данный параметр.
Можно рассматривать динамику исполнения по различным уровням бюджетов бюджетной системы субъектов РФ:
  • консолидированный бюджет субъекта;
  • бюджет субъекта;
  • местные бюджеты (консолидированный бюджет муниципальных образований);
  • муниципальные районы (по всем муниципальным районам в целом и в частности по отдельным муниципальным районам).
