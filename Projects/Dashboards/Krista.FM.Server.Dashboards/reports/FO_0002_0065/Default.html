Источник данных: данные АС Бюджет.
   Назначение: отчет предназначен для анализа сводной информации об исполнении бюджета субъекта в части реализации Федерального закона №261 в разрезе ведомств региона.
   Параметры:    выбирается дата, после выбора необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю дату, в которой производилось финансирование. В дальнейшем можно выбрать любую дату для анализа.
   При выборе параметра «тыс.руб» - отчет будет отображать суммы в тыс. руб., при выборе «млн.руб.» - в млн.руб.
   В отчете реализована возможность выгрузки информации в формат xls и pdf для последующего анализа. Каждый столбец таблицы имеет возможность сортировки, для этого необходимо нажать кнопкой мыши на наименование столбца.
   При нажатии сноски «Источники данных» выводится перечень источников, на основе которых построен отчет.
