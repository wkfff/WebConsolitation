Назначение:отчет предназначен для сравнительного анализа территорий по уровню заработной платы работников организаций и объёмам задолженности предприятий по выплате заработной платы.
   Параметры:выбирается период (дата, на которую проводится анализ данных). После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю отчетную дату, по которой имеются данные. В отчёте отображается карта рассматриваемой территории с условной раскраской в соответствии со значениями выбранного для анализа показателя. Анализ проводится по следующим показателям:
    • «Среднемесячная заработная плата в расчете на одного работника предприятий и организаций (за предшествующий период года), руб.»
    • «Суммарный объем просроченной задолженности по выплате заработной платы, тыс. руб.».
   По выбранному показателю определяются ранги территорий и выставляются звёзды-маркеры: «золотая звезда» - у территории с наилучшим значением показателя, «серая звезда» - у территории с наихудшим значением показателя.
   По тому же показателю, что отображается на карте, дополнительно выводится гистограмма, отражающая значение показателя по всем территориям в порядке возрастания значений, в сравнении с уровнем по субъекту РФ.
