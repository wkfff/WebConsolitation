Назначение: отчет предназначен для анализа информации о финансировании объектов входящих в рамки краевой адресной инвестиционной программы, с указанием плановых данных по постановлению и фактически произведенных расходов на текущую дату.
Параметры: выбирается дата, после выбора необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация на последнюю дату, в которой производилось финансирование. В дальнейшем можно выбрать любую дату для анализа, но только в рамках текущего года.
Для вывода в отчет дополнительной информации по показателям в строках, необходимо выбрать пункт в параметре «Доп.информация», после этого в таблицу будут добавлены следующие столбцы:
  • Код СубЭКР;
  • Код КЦСР;
  • Генеральный подрядчик;
  • Получатель бюджетных средств;
 Так же в области параметров можно выбрать форму построения отчета. Для выбора доступны четыре следующие формы:
  • «Общий вид» – содержит полный перечень показателей в рамках финансирования краевой адресной инвестиционной программы;
  • «Расшифровка» - содержит перечень конкретных объектов финансирования;
  • «Справка по МО» - содержит перечень конкретных объектов финансирования с разделением по муниципальным образованиям края;
  • «Газификация» - детализированная информация о финансировании объектов программы в рамках газификации края.
При выборе параметра «тыс.руб» - отчет будет отображать суммы в тыс. руб., при выборе «млн.руб.» - в млн.руб.
В таблице отчета по каждому показателю финансирования краевой адресной инвестиционной программы приводится следующая информация:
  • План краевого бюджета, исходя из последнего Постановления;
  • Фактическое финансирование с начала текущего месяца на текущую дату;
  • Фактическое финансирование с начала текущего года на текущую дату;
  • Остаток от плана;
  • Процент исполнения плана
  • План местного бюджета по софинансированию;
  • Фактическое финансирование из местного бюджета по софинансированию;
  • Процент исполнения плана местного бюджета по софинансированию.
В отчете реализована возможность выгрузки информации в формат xls и pdf для последующего анализа. Каждый столбец таблицы имеет возможность сортировки, для этого необходимо нажать кнопкой мыши на наименование столбца.
При нажатии сноски «Источники данных» выводится перечень источников, на основе которых построен отчет.
