Назначение:отчет предназначен для анализа результатов оценки качества управления финансами муниципальных образований Вологодской области по выбранному показателю.
Параметры: выбираются год, квартал (по умолчанию установлены последние, по которым имеются данные) и анализируемый показатель (по умолчанию выбирается итоговая оценка качества).
Целью проведения оценки качества управления финансами муниципальных образований является выявление негативных тенденций в организации и осуществлении бюджетным процессом в муниципальных образованиях и оперативная выработка предложений для их устранения.
Методика "Об оценке качества управления финансами муниципальных образований области"
