Назначение:отчет предназначен для анализа динамики процентных ставок по кредитам, депозитам, ипотечным и потребительским кредитам в субъекте РФ.
Параметры: выбираются:
- тип денежных средств: в рублях и в иностранной валюте;
- тип вклада (кредита): юридические лица и физические лица.
После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация по Тюменской области.
Для выбранного субъекта формируются диаграммы динамики в виде графиков по следующим показателям: «Банковские минимальные/максимальные средневзвешенные ставки по кредитам», «Банковские минимальные/максимальные средневзвешенные ставки по депозитам», «Ипотечные кредиты: процентная ставка по выданным кредитам (средневзвешенная)» и «Потребительские кредиты: процентная ставка по выданным кредитам (средневзвешенная)»
