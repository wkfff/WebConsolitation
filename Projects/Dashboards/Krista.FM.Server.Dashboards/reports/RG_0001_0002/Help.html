Источник данных:ежемесячная отчетность об исполнении бюджетов.
Назначение:отчет предназначен для анализа основных параметров исполнения бюджетов.
Параметры:выбирается год, месяц и бюджет (в том числе можно выбрать консолидированный бюджет, собственный бюджет субъекта и местные бюджеты). После изменения параметров нажмите кнопку «Обновить».
Выбранный бюджет отражается на карте. В таблицах выводятся основные параметры исполнения по доходам, расходам, дефициту и задолженности.
Также можно выбрать район (городской округ) на карте и нажать кнопку «Обновить». Будет отображаться исполнение выбранного бюджета.
На карте отображаются только те города, по которым проводится отдельное исполнение бюджета (городские округа). Остальные города скрыты.
Доступны данные с 2008 года до последнего отчетного периода. Данные пополняются ежемесячно.
Для процента исполнения выполняется цветовая оценка соответствия среднему, позволяющая оценить, какие расходные направления финансируются меньше остальных и какие доходные источники исполняются хуже прочих. По всем местным бюджетам определяется средний процент исполнения по доходам и расходам по «Доходы бюджета - ИТОГО» и «Расходы бюджета - ИТОГО» соответственно. Если процент исполнения по доходному источнику меньше среднего процента исполнения по доходам по МО, то отображается красный индикатор. Если процент исполнения по расходам меньше среднего процента исполнения по расходам по МО, то отображается красный индикатор.
