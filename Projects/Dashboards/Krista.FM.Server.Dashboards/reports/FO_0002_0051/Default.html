Назначение:отчет предназначен для анализа кредиторской задолженности.
Параметры: в параметрах отчета выбирается год, бюджет и тип средств. После выбора параметра необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний год, по которому имеются данные.
В отчете отражаются данные по кредиторской задолженности, в том числе по просроченной (нереальной к взысканию) задолженности в разрезе счетов бюджетного учета.
