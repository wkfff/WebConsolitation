Назначение:отчет предназначен для анализа уровня долговой нагрузки на бюджет автономного округа и бюджеты муниципальных образований региона, анализа просроченной задолженности по государственному и муниципальному долгу.
Параметры: выбирается месяц и год. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний отчетный период.
В таблице в разрезе бюджетов приводятся следующие показатели:
  • объем долга – для строчки «Бюджет субъекта» выводится государственный долг, для муниципального образования выводится сумма муниципального долга;
  • исполнение по доходам (в соответствии с уровнем бюджета в строках) - выводится общий объем доходов за исключением субвенций;
  • уровень долговой нагрузки с индикацией минимального (желтая звезда) и максимального значения (серая звезда) среди бюджетов;
  • темп роста долговой нагрузки к прошлому году;
  • просроченная задолженность по долговым обязательствам - для строчки «Бюджет субъекта» выводится просроченная задолженность по государственному долгу, для муниципального образования выводится просроченная задолженность по муниципальному долгу;
  • удельный вес просроченной задолженности по долговым обязательствам в общем объеме долговых обязательств.
Показатель «Уровень долговой нагрузки» рассчитывается в соответствии с Приказом Минфина РФ от 3 декабря 2010 г. №552 «О порядке проведения оценки качества организации и осуществления бюджетного процесса в субъектах РФ» рассчитывается как отношение объема государственного (муниципального) долга к доходам бюджета субъекта (муниципального образования) за исключением субвенций.
Картограмма отражает уровень долговой нагрузки по всем муниципальным образованиям на выбранный период.
