Назначение:отчет предназначен для анализа динамики и структуры основных показателей, характеризующих состояние окружающей среды в МО и меры по ее охране.
   Параметры:нет.
   В отчете выводится информация по годам, по которым имеются данные в системе (в некоторых элементах значения показателей выводятся по состоянию на последний год, по которому есть данные).
   В таблице в верхней части отчета представлены данные о выбросах загрязняющих веществ в атмосферу (по всему объему и отдельно по жидким и газообразным веществам) за несколько лет.
   На диаграммах справа от таблицы можно просмотреть структуру каждого показателя из таблицы по видам загрязняющих веществ за год, выбранный в таблице по строкам.
   В таблице ниже представлены данные по основным показателям состояния окружающей среды в МО и ее охраны за последний год, по которому есть данные. Динамику выбранного в таблице показателя за несколько лет можно просмотреть на диаграмме справа.
