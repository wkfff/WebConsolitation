Назначение: отчет предназначен для анализа динамики и структуры основных показателей, характеризующих территорию МО.

Параметры: нет.

В отчете выводится информация по годам, по которым имеются данные (в некоторых элементах значения показателей выводятся по состоянию на последний год, по которому есть данные в системе).

В верхней части отчета представлены общие сведения о территории МО и ее использовании по состоянию на последний год, по которому есть данные в системе.

Далее в таблице представлены данные по основным показателям, характеризующим территорию МО, за несколько лет. Структуру каждого показателя за год, выбранный в таблице по столбцам, можно просмотреть на диаграммах под таблицей.

В таблице ниже отображены данные за последний год по показателям земельного контроля и регулирования застройки в МО. Динамику выбранного в таблице показателя можно просмотреть на диаграмме справа.

В таблице в нижней части отчета представлены данные площади застроенных земель за ряд лет. Структурные диаграммы справа от таблицы отражают распределение данной площади по видам объектов застройки за год, выбранный в таблице по строкам.
