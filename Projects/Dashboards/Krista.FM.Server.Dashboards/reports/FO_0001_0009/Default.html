Источник данных:отчетность об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, областного и местных бюджетов, база планирования бюджета.
   Назначение:отчет предназначен для анализа структуры общей суммы доходов и структуры налоговых доходов.
   Параметры:
Выбирается одна из опций:
a. «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
b. «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
c. «Проект бюджета» - в отчет выводятся данные проектов бюджетов по доходам и расходам.
Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» «Месяц» (по умолчанию установлены последние, по которым имеются данные), «Бюджет» (можно выбрать консолидированный бюджет, бюджет субъекта и местные бюджеты).
Для опции «Проект бюджета» в отчет выводятся параметры:
d. «Вариант доходов» - выбирается год, который является текущим, и вариант проекта доходов в указанном году;
e. «Бюджет» - можно выбрать консолидированный бюджет, бюджет субъекта и местные бюджеты. После изменения параметро нажмите кнопку «Обновить».
При выборе муниципального района появляются дополнительные параметры «Консолидированный бюджет МР», «Бюджет муниципального района», «Свод бюджетов поселений». По умолчанию в отчете отображаются данные по консолидированному бюджету района. Чтобы посмотреть данные по собственному бюджету муниципального района, бюджетам поселений необходимо в параметрах выбрать соответственно «Бюджет муниципального района», «Свод бюджетов поселений».
В таблице по соответствующему уровню бюджета отображаются:
- Сумма фактического исполнения (плановых назначений, проектных данных) по соответствующим доходным источникам.
- Доля доходного источника (группы доходных источников) в общей сумме доходов бюджета.
По выбранному периоду и уровню бюджета отображаются две диаграммы в нижней части отчета.
Диаграмма слева позволяет оценить структуру фактических (плановых, проектных) данных по доходам в целом. Диаграмма справа позволяет сравнить структуру фактических (плановых, проектных) данных по налоговым доходам.
