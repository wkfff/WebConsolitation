Источник данных:общедоступные данные, опубликованные Федеральным казначейством (Казначейством России) http://roskazna.ru/reports/mb.html.
Назначение:отчет предназначен для сравнительного анализа исполнения по доходам и среднедушевых доходов субъектов РФ.
Параметры:выбираются год и месяц (по умолчанию установлены последние, по которым имеются данные), федеральный округ и вид дохода. После изменения параметров нажмите кнопку «Обновить».
В таблице по каждому субъекту приводятся:
  • Фактическое исполнение по выбранному виду доходов;
  • Плановые годовые назначения;
  • Процент исполнения плана с цветовой оценкой равномерности исполнения (1/12 годового плана в месяц);
  • Ранг по проценту исполнения в федеральном округе (лучший субъект в федеральном округе отмечается золотой звездой, худший – серой);
  • Ранг по проценту исполнения в РФ (лучший субъект отмечается золотой звездой, худший – серой);
  • Численность постоянного населения;
  • Среднедушевые доходы по выбранному виду доходов;
  • Ранг по среднедушевым доходам в федеральном округе (лучший субъект в федеральном округе отмечается золотой звездой, худший – серой);
  • Ранг по среднедушевым доходам в РФ (лучший субъект отмечается золотой звездой, худший – серой);
  • Доля (вклад) данного субъекта в доходы федерального округа;
  • Доля (вклад) данного субъекта в доходы всех субъектов РФ.
Если выбраны все федеральные округа, то выводится карта РФ по федеральным округам. Если выбран один федеральный округ, то выводится карта выбранного федерального округа по субъектам РФ.
На карте заливкой отображается процент исполнения плана по выбранному доходному источнику.
