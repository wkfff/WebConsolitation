Источник данных:отчетность об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, областного бюджета и местных бюджетов, база планирования бюджета.
   Назначение:отчет предназначен для анализа распределения бюджетных расходов между областным и местными бюджетами.
   Параметры:
Выбирается одна из опций:
a. «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
b. «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
c. «Проект бюджета» - в отчет выводятся данные проектов бюджетов по доходам и расходам.
Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» «Месяц» (по умолчанию установлены последние, по которым имеются данные).
Для опции «Проект бюджета» в отчет выводится параметр «Год» «Месяц» и «Вариант расходов» - вариант проекта расходов в указанном году.
После изменения параметров нажмите кнопку «Обновить».
Сопоставление проводится по основным расходным статьям областного и местных бюджетов. За 100 процентов берутся расходы по соответствующим статьям консолидированного бюджета.
В таблице по соответствующему уровню бюджета отображаются:
- Сумма фактического исполнения (бюджетных назначений, проектные данные) по соответствующим расходам.
- Доля статьи расходов (группы статей) в общей сумме расходов консолидированного бюджета.
- Суммы по статьям расходов (группам статей), исключаемые из консолидированного бюджета (при выборе данных с 2011 года).
По выбранным данным отображается линейчатая диаграмма с накоплением.
