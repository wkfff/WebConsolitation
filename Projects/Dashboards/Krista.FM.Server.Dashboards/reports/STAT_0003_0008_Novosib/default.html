Источник данных:данные ежемесячного мониторинга начисленной среднемесячной заработной плате в расчете на одного работника предприятий и организаций по видам экономической деятельности.
   Назначение:отчет предназначен для анализа размеров начисленной среднемесячной заработной плате в расчете на одного работника предприятий и организаций по видам экономической деятельности в среднем за год.
   Параметры:выбираются года для анализа и год для сравнения. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последние три года, по которым имеются данные, и производится сравнение с предыдущим годом.
   В таблице выводятся:
   • значение показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» в разрезе видов деятельности в среднем за год;
   • темп роста значения показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» в среднем ха год с аналогичным показателем за выбранный для сравнения год;
   • значение показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» по видам деятельности в среднем за год в сравнении со среднеобластным уровнем этого же показателя.
   Для показателей отображается визуальная индикация. Прирост значения показателя иллюстрирует зелёная стрелка вверх, снижение – красная стрелка вниз. Значение выше среднеобластного уровня иллюстрируется зелёный круг, значение ниже среднеобластного уровня – красный круг.
