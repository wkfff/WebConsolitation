Источник данных:ежедневная информация об исполнении кассового плана окружного бюджета.
Назначение:отчет предназначен для анализа динамики и структурной динамики исполнения кассового плана окружного бюджета.
Параметры:год, квартал и показатель кассового плана. После изменения параметра нажмите кнопку «Обновить». По умолчанию данные выводятся за текущий квартал.
В отчете по выбранному показателю кассового плана выводится на диаграмме динамика исполнения за квартал.
