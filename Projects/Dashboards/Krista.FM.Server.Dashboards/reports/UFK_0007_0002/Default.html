Источник данных:данные, передаваемые управлением Федерального казначейства по Новосибирской области, в соответствии с приложением №16 «Сводная ведомость по кассовым поступлениям в бюджеты (месячная)» к приказу Федерального казначейства РФ от 10 октября 2008 г. N 8н.
 Назначение:предназначен для анализа темпа роста доходов бюджетов муниципальных образований к аналогичному периоду предыдущего года.
Параметры:выбираются год и месяц. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводиться информация за последний период, за который имеются данные.
В таблице по каждому виду доходов отображается:
  - фактическое исполнение с начала года;
  - фактическое исполнение за аналогичный период прошлого года;
  - темп роста доходов к аналогичному периоду прошлого года (стрелками показывается рост/сокращение доходов по сравнению с прошлым годом);
  - доля данного вида доходов муниципального образования в общей сумме доходов бюджета муниципальных образований для оценки значимости доходного источника;
  - доля данного вида доходов муниципального образования в общей сумме доходов бюджета муниципальных образований в прошлом году.
