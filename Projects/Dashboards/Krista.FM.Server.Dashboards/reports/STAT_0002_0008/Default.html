Назначение:отчет предназначен для анализа объемов выданных кредитов в разрезе банков Ханты-Мансийского автономного округа - Югры.
Параметры: выбирается период. После выбора параметра необходимо нажать кнопку «Обновить». По умолчанию, в параметре автоматически установлен последний период, на который есть данные в системе.
В таблице в разрезе банков субъекта РФ приведены значения показателей банковской деятельности за выбранный в параметре период. Также в таблице отображена динамика показателей за период: рассчитаны абсолютные отклонения и темпы прироста. Для прямых показателей прирост значения иллюстрирует зеленая стрелка вверх, а его снижение – красная стрелка вниз. Для обратных показателей прирост значения иллюстрирует красная стрелка вверх, а его снижение – зеленая стрелка вниз.
На основании данных таблицы строится диаграмма распределения предприятий банковского сектора по общему объему выданных ипотечных и потребительских кредитов.
