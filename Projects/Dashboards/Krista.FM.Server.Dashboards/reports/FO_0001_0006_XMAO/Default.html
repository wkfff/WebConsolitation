Назначение отчета: отчет предназначен для отражения утвержденной штатной численности государственных гражданских служащих и работников, замещающих должности муниципальной службы в разрезе категорий и групп должностей по субъекту РФ.

Параметры: выбирается год. После выбора параметра необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний год, по которому имеются данные.

Таблица отчета содержит данные об утвержденной штатной численности по отдельным государственным органам (областная Дума, Администрация области, Избирательная комиссия) и по исполнительным органам государственной власти, а так же об утвержденной штатной численности работников, замещающих должности муниципальной службы.

Данные по показателю «Утвержденная штатная численность» отражаются по гр. 3 формы 14 и формы 14 МО.
