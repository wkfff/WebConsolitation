Источник данных: Общедоступные данные, опубликованные Федеральным казначейством (Казначейством России).

Назначение: Отчет предназначен для сравнения исполнения бюджетов субъектов по показателям доходов, расходов, дефицита, профицита и источников финансирования.

Параметры: выбираются год, месяц и субъекты, которые необходимо сравнить. После изменения параметров нажмите кнопку «Обновить».
В отчете по выбранным субъектам сравниваются показатели за несколько лет по консолидированному бюджету и бюджету субъекта.
