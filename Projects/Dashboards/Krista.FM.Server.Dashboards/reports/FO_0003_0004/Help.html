Назначение:отчет предназначен для анализа динамики поступлений доходов в сравнении с предшествующими годами в помесячной и поквартальной разбивке.
Параметры:выбираются годы, по которым проводится сравнение (по умолчанию три последних года), уровень бюджета и вид дохода. После изменения параметров нажмите кнопку «Обновить». По умолчанию в отчете выводятся фактические поступления доходов за указанный период. Для того чтобы анализировать сумму поступлений нарастающим итогом с начала года, необходимо поставить галку в параметре «Сумма с начала года с накоплением».
На диаграмме отчета по умолчанию показана помесячная динамика поступлений доходов. Если необходим анализ поквартальных поступлений, то необходимо поставить галку в параметре «Поквартальная динамика».
При выборе параметра муниципальные районы или определенного муниципального района появляется дополнительный параметр «Консолидированный бюджет МО». По умолчанию этот параметр включен и в отчете отображаются данные с учетом консолидированного бюджета районов. Чтобы посмотреть данные с учетом бюджетов муниципальных районов, необходимо отключить данный параметр, т.е. убрать галочку.
В таблицах отчета приводиться поквартальная и помесячная оценка доходов по следующим показателям:
  • «Факт» - по умолчанию выводиться сумма поступлений за период. Если выбрать параметр «Сумма с начала года с накоплением», то будет выведена сумма фактических поступлений нарастающим итогом с начала года;
  • «Темп роста» - выводиться темп роста показателя «Факт» к аналогичному периоду прошлого года. Показатель имеет цветовую индикацию стрелками: зеленой – если прирост к прошлому году, красной – если снижение поступлений к прошлому году.
