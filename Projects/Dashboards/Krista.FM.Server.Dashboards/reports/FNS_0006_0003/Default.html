Источник данных:отчетность Федеральной налоговой службы о налоговой базе и структуре начислений по местным налогам (форма № 5-МН), отчетность Федеральной налоговой службы о налоговой базе и структуре начислений по налогу на прибыль организаций (форма № 5-ПМ), отчетность Федеральной налоговой службы о налоговой базе и структуре начислений по налогу на имущество организаций (форма № 5-НИО), отчетность Федеральной налоговой службы о структуре начислений по транспортному налогу (форма № 5-ТН). Формы № 5-МН, № 5-ПМ, № 5-НИО, № 5-ТН утверждены приказом ФНС Росси от 30.12.2008 №ММ-3-1/693@.
   Назначение:Отчет предназначен для анализа льгот, предоставляемых в соответствии с федеральным, региональным и местным законодательством.
   Параметры:
Управляющим параметром является «Год». После выбора года необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний период, по которому есть данные в базе.
Информация приводится по следующим налогам:
- налог на прибыль
- налог на имущество физических лиц
- налог на имущество организаций
- транспортный налог
- земельный налог
По каждому из данных налогов приводится сумма налога, подлежащая уплате в бюджет, объем предоставляемых льгот, и доля льгот в сумме исчисленного к уплате налога.
При выделении одного из налогов в таблице, по нему строится круговая диаграмма, отображающая объем льгот по данному налогу предоставляемый в соответствии с федеральным, региональным и местным законодательством. Также в отчете представлена общая диаграмма, на которой по всем налогам отображается сумма налога, подлежащая уплате в бюджет и объем предоставляемых льгот.
