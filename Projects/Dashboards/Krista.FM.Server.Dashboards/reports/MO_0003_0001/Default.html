Назначение:отчет предназначен для анализа социально-экономического положения муниципального образования субъекта РФ.
Параметры:выбираются следующие параметры:
  • муниципальное образование;
  • показатели паспорта МО;
  • год;
  • периодичность.
После выбора параметров необходимо нажать на кнопку «Обновить».
Отчет состоит из двух разделов: Общее описание, квартальные или годовые данные (по необходимости можно скрыть раздел общие сведения). Раздел 2 представляет собой таблицу, сформированную по выбранным параметрам. В отчете реализован множественный выбор параметров. Помимо значений за выбранный период в таблице считаются прирост и темп роста значений показателей по отношению к предыдущему периоду. Для наглядности в таблице приведена индикация:
  • для прямых показателей: зеленая стрелка вверх иллюстрирует положительный темп роста, красная стрелка вниз – отрицательный темп роста;
  • для обратных показателей: красная стрелка вверх иллюстрирует положительный темп роста, зеленая стрелка вниз – отрицательный темп роста.
