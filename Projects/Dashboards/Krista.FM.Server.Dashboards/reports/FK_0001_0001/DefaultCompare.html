Источник данных:
  • ежемесячные отчеты об исполнении бюджетов субъектов РФ в формате html, размещаемые на сайте федерального казначейства РФ (http://www.roskazna.ru/reports/mb.html);
  • данные долговых книг субъектов размещаемые на сайте Министерства финансов РФ (http://www1.minfin.ru/ru/public_debt/subdbt/).
 Назначение:отчет предназначен для сравнения структуры расходов субъектов РФ по разделам классификации расходов, сравнения плановой и фактической структуры расходов.
Параметры:выбираются год и месяц (по умолчанию установлен последний месяц, по которым имеются данные), федеральный округ и уровень бюджета. После изменения параметров нажмите кнопку «Обновить».
В таблице по выбранному субъекту по каждому разделу классификации расходов отображается:
  • фактическое исполнение с начала года;
  • доля данного вида расходов в общей сумме расходов бюджета;
  • ранг (место) по доле расхода среди субъектов федерального округа (субъект с наибольшей долей отмечается желтой звездой, субъект с наименьшей – серой звездой);
  • ранг (место) по доле расхода среди всех субъектов (субъект с наибольшей долей отмечается желтой звездой, субъект с наименьшей – серой звездой).
По выбранной в таблице строке отображаются две диаграммы в нижней части отчета. При выборе другого субъекта изменяются диаграммы.
Диаграмма слева позволяет сравнить фактическую и плановую структуру расходов.
Диаграмма справа позволяет сравнить структуру расходов по субъектам федерального округа.
Используемые аббревиатуры и сокращения:
ЖКХ – жилищно-коммунальное хозяйство;
МБТ – межбюджетные трансферты;
МО – муниципальное образование;
РФ – Российская Федерация;
СМИ – средства массовой информации;
ФО – федеральный округ.
