Назначение: отчет предназначен для анализа сбалансированности бюджетов поселений.

Параметры: выбираются год, месяц (по умолчанию последний, по которому имеются данные) и муниципальный район (по умолчанию первый в списке).

После выбора параметра необходимо нажать на кнопку «Обновить».

Содержание:

В таблице отчета по выбранным параметрам должны отображаться следующие данные:

  • общее количество поселений района;
  • количество профицитных бюджетов поселений района по плановым данным и по фактическому исполнению;
  • количество дефицитных бюджетов поселений района по плановым данным и по фактическому исполнению;
  • количество сбалансированных бюджетов поселений района по плановым данным и по фактическому исполнению;
  • общий объем профицита бюджетов поселений по плановым данным и по фактическому исполнению;
  • общий объем дефицита бюджетов поселений по плановым данным и по фактическому исполнению.

На гистограмме представлено распределение поселений выбранного в параметре района. Поселения выводятся на диаграмме в порядке убывания суммы дефицита бюджета и возрастания профицита. Первым выводится поселение с самым большим дефицитом бюджета, последним – поселение с самым большим профицитом. С помощью переключателей «План» и «Факт» можно выводить на диаграмму уточненные годовые назначения или суммы фактического дефицита/профицита.

Данные в отчете могут выводиться как в тысячах, так и в миллионах рублей.
