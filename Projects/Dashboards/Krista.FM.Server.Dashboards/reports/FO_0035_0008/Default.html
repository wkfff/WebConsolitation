Источник данных:данные АС «Бюджет».
Назначение:анализ исполнения графика финансирования кассовых выплат из областного бюджета.
Параметры:Дата. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний день, по которому имеются данные.
В отчете представлены данные по сводной бюджетной росписи, график финансирования на текущий месяц, кассовые выплаты за месяц и остаток графика финансирования в разрезе расходных показателей.
Данные представлены в тысячах рублей.
