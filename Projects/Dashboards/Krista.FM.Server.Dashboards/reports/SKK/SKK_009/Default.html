Назначение: отчет предназначен для анализа досмотренных лиц на выбранной территории.

Параметры: выбираются следующие параметры:

  • территория, по умолчанию – РФ;
  • отчетный период (год, месяцы), по умолчанию - последний месяц, по которому имеются данные;
  • период для сравнения (год, месяцы), по умолчанию - предпоследний месяц, по которому имеются данные;
  • направление, по умолчанию - Всего.

После выбора параметров необходимо нажать на кнопку «Обновить».

Содержание: по выбранному участку Государственной границы РФ строятся:

  • таблица по выбранной территории – абсолютные (досмотрено лиц, выявлено больных) и относительные показатели (выявлено больных) по текущему периоду, рост/снижение показателя (выявлено) в сравнении с предыдущим периодом;
  • круговая диаграмма по выбранной территории – для каждого сектора отображаются абсолютные значения (число досмотренных лиц) и доля в %. Диаграмма строится - в разрезе ФО, если в параметрах отчета выбрана РФ; в разрезе субъектов, если выбран ФО; в разрезе пунктов пропуска, если выбран субъект;
  • столбчатая диаграмма по выбранной территории – абсолютные значения в сумме (досмотрено лиц, выявлено больных);
  • столбчатая перевернутая диаграмма по участкам Государственной границы РФ, которые проходят через выбранную территорию – абсолютное значение (досмотрено лиц) и доля в %;
  • таблица по видам сообщения, которые есть на выбранной территории – абсолютные значения по пунктам пропуска (досмотр лиц, выявлено больных);
  • круговая диаграмма по видам сообщения, которые есть на выбранной территории – абсолютные значения (число лиц) и доля в %.
