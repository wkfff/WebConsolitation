Назначение: отчет предназначен для анализа результатов санитарно-карантинного контроля в выбранном пункте пропуска через Государственную границу РФ.

Параметры: выбираются следующие параметры:

  • отчетный период (год, месяцы), по умолчанию - последний месяц, по которому имеются данные;
  • участок Государственной границы РФ, по умолчанию – первый в списке.

После выбора параметров необходимо нажать на кнопку «Обновить».

Содержание: справка о результатах санитарно-карантинного контроля через выбранный участок Государственной границы РФ.
