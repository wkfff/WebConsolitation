Назначение: отчет предназначен для анализа данных санитарно-карантинного контроля на выбранной территории.

Параметры: выбираются следующие параметры:

  • территория, по умолчанию – РФ;
  • отчетный период (год, месяцы), по умолчанию - последний месяц, по которому имеются данные;
  • период для сравнения (год, месяцы), по умолчанию - предпоследний месяц, по которому имеются данные;
  • направление, по умолчанию - Всего;
  • вид сообщения, по умолчанию – Всего;
  • детализировать по пунктам пропуска, по умолчанию – отключен.

После выбора параметров необходимо нажать на кнопку «Обновить».

Содержание: по выбранной территории строится таблица - значения абсолютных и относительных показателей, а также рост/снижение показателя текущего периода по сравнению с предыдущим.
Рост/снижение выводится в процентах, если значение не превышает 100 %, в разах, если значение превышает 100 %, в абсолютных значениях, если в одном из периодов (текущем или предыдущем) значение показателя равно нулю. Дополнительно индикаторы-стрелки показывают факт роста/снижения данного показателя.
Индикаторы показывают наибольшее и наименьшее значение в столбце относительного показателя.
Формирование строк таблицы зависит от параметров «Детализировать по пунктам пропуска» и «Территория». Если параметр «Детализировать по пунктам пропуска» включен, то в строки к субъектам выводятся пункты пропуска. Столбец 2 «Число пунктов пропуска, в которых осуществлялся СКК» заполняется значением «1» в строках, соответствующих одному пункту пропуска, при включенном параметре «Детализировать по пунктам пропуска».
