Назначение: отчет предназначен для анализа досмотренных транспортных средств на выбранном участке Государственной границы РФ.

Параметры: выбираются следующие параметры:

  • участок Государственной границы РФ, по умолчанию – Всего;
  • отчетный период (год, месяцы), по умолчанию - последний месяц, по которому имеются данные;
  • период для сравнения (год, месяцы), по умолчанию - предпоследний месяц, по которому имеются данные;
  • направление, по умолчанию - Всего.

После выбора параметров необходимо нажать на кнопку «Обновить».

Содержание: по выбранному участку Государственной границы РФ строятся:

  • таблица – абсолютные и относительные показатели (досмотрено ТС, приостановлено ТС) текущего и предыдущего периода, рост/снижение показателя (приостановлено) в сравнении с предыдущим периодом. Таблица строится в разрезе участков Государственной границы РФ (один или все – в зависимости от выбранного значения параметра), субъектов РФ и пунктов пропуска;
  • круговая диаграмма - если в параметрах отчета выбран участок Государственной границы РФ, то диаграмма строится в разрезе пунктов пропуска; если участок Государственной границы РФ не выбран, то диаграмма строится в разрезе ФО. Для каждого сектора выводится абсолютное значение (число досмотренных ТС) и доля в %;
  • столбчатая диаграмма – выводится в сумме по выбранному участку Государственной границы РФ в абсолютных значениях;
  • таблица по видам сообщения - в таблицу включаются только виды сообщения, которые есть на выбранном участке Государственной границы РФ. Если вид сообщения один, то таблица должна состоять из одной строки, соответствующей виду сообщения;
  • круговая диаграмма по видам сообщения - в диаграмму включаются только виды сообщения, которые есть на выбранном участке Государственной границы РФ. Если вид сообщения один, то диаграмма не выводится. Для каждого сектора отображаются абсолютные значения (число досмотренных ТС) и доля в %.
