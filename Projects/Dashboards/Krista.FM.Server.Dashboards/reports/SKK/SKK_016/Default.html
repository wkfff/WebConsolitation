Назначение: отчет предназначен для анализа транспортных средств на выбранном пункте пропуска.

Параметры: выбираются следующие параметры:

  • пункт пропуска, по умолчанию - первый в списке пунктов пропуска;
  • отчетный период (год, месяцы), по умолчанию - последний месяц, по которому имеются данные;
  • период для сравнения (год, месяцы), по умолчанию - предпоследний месяц, по которому имеются данные;
  • направление, по умолчанию - Всего.

После выбора параметров необходимо нажать на кнопку «Обновить».

Содержание: по выбранному пункту пропуска строятся:

  • таблица – абсолютные и относительные показатели (досмотрено ТС, приостановлено ТС) текущего и предыдущего периода, рост/снижение показателя (приостановлено) в сравнении с предыдущим периодом;
  • столбчатая диаграмма – абсолютные значения (досмотрено ТС, приостановлено ТС).
