Источник данных:ежемесячные отчеты об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, собственного бюджета субъекта РФ и местных бюджетов.
Назначение:отчет предназначен для анализа темпа роста (падения) доходов к аналогичному периоду предыдущего года. Для оценки важности доходного источника приведена доля вида доходов в бюджете.
Параметры:выбираются год и месяц (по умолчанию установлены последние, по которым имеются данные). После изменения параметров нажмите кнопку «Обновить».
При установке галочки в параметре «Природно-климатические зоны», перечень муниципальных районов в таблице автоматически распределится на природно-климатические зоны.
В таблице по выбранному бюджету по каждому виду доходов отображается:
  • фактическое исполнение с начала года;
  • фактическое исполнение за аналогичный период прошлого года;
  • темп роста доходов к аналогичному периоду прошлого года (более 100% = рост = зеленая стрелка вверх, менее 100% = снижение = красная стрелка вниз);
  • ранг муниципального района или зоны по темпу роста (желтая звезда – наибольший темп роста, серая звезда – минимальный рост);
  • доля данного вида дохода в общей сумме доходов бюджета для оценки значимости доходного источника (стрелкой отображается рост/падение доли по сравнению с долей в прошлом году);
  • доля данного вида дохода в общей сумме доходов бюджета в прошлом году.
 По налоговым и неналоговым доходам и НДФЛ в таблице дополнительно представлен расчет следующих показателей:
  • фактическое исполнение в прошлом году с учетом изменения дополнительного норматива по НДФЛ на текущий год для муниципальных районов;
  • темп роста к прошлому году с учетом изменения дополнительного норматива по НДФЛ на текущий год для муниципальных районов;
  • ранг МР по темпу роста с учетом изменения дополнительного норматива по НДФЛ на текущий год для муниципальных районов.
