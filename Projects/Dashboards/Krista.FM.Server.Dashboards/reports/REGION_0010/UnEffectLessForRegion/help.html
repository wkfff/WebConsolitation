Назначение:отчет предназначен для оценки неэффективных расходов по отдельным сферам и по выбранному муниципальному образованию.
Параметры:в отчете применяются параметры:
    - Год. Используется для выбора отчетного года;
    - Территория. Используется для выбора муниципального образования ХМАО-Югры.
    - Сфера. Используется для выбора сферы неэффективных расходов (здравоохранение, общее образование, ЖКХ, организация муниципального управления);
    В таблице выводятся основные показатели оценки неэффективных расходов. Доля неэффективных расходов рассчитывается как отношение объема неэффективных расходов на сферу к общему объему расходов бюджета муниципального образования на выбранную сферу. Для каждого муниципального образования по основным показателям рассчитывается ранг по ХМАО-Югре. Применена маркировка стрелками для визуального отображения динамики показателей.
    В отчете выбранного муниципального образования и по выбранной сфере строятся диаграммы: «Доля неэффективных расходов в общем объеме расходов на сферу» и «Объем неэффективных расходов на сферу».
    Для сфер «Здравоохранение» и «Общее образование» в отчете присутствует параметр «Детализация», при выборе которого в отчете открываются таблицы детализации по неэффективным расходам с данными за последние отчетный и предыдущий года. При первоначальном открытии таблицы детализации скрыты.
