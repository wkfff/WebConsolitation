Назначение:отчет предназначен для анализа размеров начисленной среднемесячной заработной плате в расчете на одного работника предприятий и организаций по видам экономической деятельности.
   Параметры:выбирается отчётный период, за который предоставляются данные по показателям, и год, с соответствующим периодом которого проводится сравнение. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний отчетный период, по которому имеются данные, и производится сравнение с предыдущим годом, по которому есть данные.    
    В таблице выводятся:
  • значение показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» в разрезе видов деятельности за последний отчётный период;
  • значение показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» в процентах от значения за соответствующий отчётный период выбранного для сравнения года;
  • значение показателя «Начисленная среднемесячная заработная плата в расчете на одного работника предприятий и организаций» по видам деятельности в сравнении со среднеобластным уровнем.
   Для показателей отображается визуальная индикация. Прирост значения показателя иллюстрирует зелёная стрелка вверх, снижение – красная стрелка вниз. Значение выше среднеобластного уровня иллюстрируется зелёный круг, ниже среднеобластного уровня – красный круг, равенство со среднеобластным уровнем – жёлтый круг.
   На диаграмме отражается сравнительная характеристика размера заработной платы и среднесписочной численности работников по основным видам экономической деятельности.
