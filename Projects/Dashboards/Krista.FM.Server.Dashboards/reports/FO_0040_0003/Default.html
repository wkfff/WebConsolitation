Назначение:отчет предназначен для анализа результатов мониторинга качества управления бюджетным процессом и соблюдения требований бюджетного законодательства РФ муниципальными образованиями Самарской области по выбранному показателю.
Параметры: выбираются год, квартал (по умолчанию установлены последние, по которым имеются данные) и анализируемый показатель (по умолчанию выбирается рейтинг).
На карте Самарской области представлены результаты оценки выбранного показателя за выбранный период.
