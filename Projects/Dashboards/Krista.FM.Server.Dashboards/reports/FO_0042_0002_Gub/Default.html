Назначение:отчет предназначен для анализа результатов оценки качества финансового менеджмента, осуществляемого главными распорядителями бюджетных средств в муниципальном образовании город Губкинский.
   Параметры:выбираются год, полугодие (по умолчанию установлен последние, по которому имеются данные).
   Оценка качества финансового менеджмента, осуществляемого главными распорядителями бюджетных средств в муниципальном образовании город Губкинский, осуществляется с 2010 года.
