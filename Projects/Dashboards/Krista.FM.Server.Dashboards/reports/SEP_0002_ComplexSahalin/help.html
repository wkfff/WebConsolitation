Назначение:в отчете представлен перечень исходных показателей сфер, которые используются для расчета комплексной оценки социально-экономического развития.
   Параметры:
  • Показатель;
  • Период.
   Описание:в таблицах отображаются исходные для расчета комплексной оценки социально-экономического развития в разрезе муниципальных образований. В параметре «Показатель» есть следующие группы показателей:
  • исходные показатели социальной сферы;
  • исходные показатели экономической сферы;
  • исходные показатели финансовой сферы;
  • исходные показатели оценки населением ситуации в ключевых сферах деятельности органов местного самоуправления.
   В группах исходных показателей по сферам можно выбрать отдельный показатель. При выборе одного исходного показателя в сфере в отчете строится гистограмма по всем МО по выбранному показателю в порядке убывания значений и картограмма.
