Назначение отчета:отчет предназначен для анализа уровня долговой нагрузки на местные бюджеты.
Параметры: выбираются год, месяц и уровень бюджета. После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводиться информация по консолидированному бюджету субъекта за последний период, за который имеются данные.
В таблице приводятся следующие данные:
  • сумма государственного (муниципального) долга;
  • доля объема долговых обязательств каждого местного бюджета в общем объеме муниципального долга;
  • общая сумма доходов бюджета;
  • сумма безвозмездных поступлений;
  • сумма доходов без учета безвозмездных поступлений;
  • отношение долга к доходам без учета безвозмездных поступлений;
  • ранг бюджета по уровню долговой нагрузки;
  • изменение государственного (муниципального) долга к началу года и к аналогичному периоду прошлого года.
Индикацией отражается соблюдение норматива отношения долга к доходам без учета безвозмездных поступлений (<=100% - норматив соблюдается, >100 % - норматив не соблюдается). Также индикация отражает бюджеты, имеющие максимальный и минимальный уровни долговой нагрузки.
