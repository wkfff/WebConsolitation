Назначение:в отчете рассчитывается интегральный показатель уровня жизни населения, который отражает количественное изменение социально-экономического состояния территории через изменение ограниченного числа параметров, характеризующих основные сферы жизни населения.
   Параметры:в отчете выбираются следующие параметры:
  1. Год;
  2. Интегральный показатель по каждой сфере жизни населения;
  3. Значения показателей каждой сферы жизни населения.
   После выбора параметров необходимо нажать на кнопку «Обновить». По умолчанию в отчете выводится информация об интегральном показателе по всем сферам жизни населения за последний отчетный год, по которому есть данные в системе.
   Если в параметре 2 выбирается интегральный показатель, то в таблице выводится интегральный показатель в целом по всем сферам жизни населения, а также интегральный показатель по каждой сфере в отдельности. Если же в параметре выбирается сфера жизни населения, то таблица строится по показателям выбранной сферы. Для каждого показателя выводится значение, прирост к предыдущему году, темп прироста к предыдущему году, который сопровождается маркировкой: для прямых показателей увеличение темпа прироста иллюстрирует зеленая стрелка вверх, снижение - красная стрелка вниз; для обратных показателей увеличение темпа прироста иллюстрирует красная стрелка вверх, снижение - зеленая стрелка вниз, также выводится ранг по всем муниципальным образованиям. Первый и последний ранг иллюстрирует соответственно желтая и серая звезда.
   В нижней части отчета по выбранному значению параметра 3 строится диаграмма и карта, которые характеризуют распределение муниципальных образований по выбранному показателю параметра. На диаграмме также можно сравнить интегральный показатель со средним по МО и с медианой.
