Источник данных:данные о заключённых контрактах на закупку в разрезе основных видов продукции – товаров, услуг, работ – по анализируемой территории.
   Назначение:отчет предназначен для анализа динамики и структуры государственного заказа в рамках отдельных видов закупаемой продукции за определенный период времени как в целом по всем заказчикам, так и по каждому из них.
   Параметры:в качестве параметра выбирается период анализа:
   • год;
   • квартал.
   После выбора параметров необходимо нажать кнопку «Обновить».
   По умолчанию в отчете выводится информация в целом по всем заказчикам за последний отчётный квартал.
   Для выбранного периода формируется диаграмма структуры заключённых контрактов по основным видам продукции – товары, услуги, работы – по всем заказчикам, и диаграммы структурной динамики в разрезе основных видов продукции и видов продукции согласно ОКДП.
   Таблица строится по четырём последним кварталам путём отсчёта от выбранного в параметре, который выводится последним. Диаграммы по умолчанию формируются по всем заказчикам. Для того, чтобы данные на диаграмме отобразились по одному конкретному заказчику, необходимо в таблице встать на строку с нужным заказчиком.
