Источник данных: данные мониторинга о прекращении действия контрактов.
Назначение: отчёт предназначен для мониторинга количества расторгнутых контрактов и анализа фактического их исполнения за определенный период времени, за который предоставляются сведения о контрактах.
Параметры: для фильтрации данных отчёта введены следующие параметры:
  • Год: период предоставления сведений об исполнении контрактов;
  • Заказчик: выбор заказчика, сведения о контрактах которого требуется получить.
После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчёте отображаются данные о контрактах за последний отчётный год по всем заказчикам.
