Назначение:отчет предназначен для мониторинга исполнения количественных значений индикаторов результативности программы социально-экономического развития Сахалинской области.
   Параметры:Отчет состоит из двух элементов: таблица и диаграмма.
   Параметрами влияющими на таблицу являются «Период» и «Направление». Период – отчетный год, за который производится оценка индикаторов. Направление – перечень индикаторов характеризующих направление программы социально-экономического развития Сахалинской области. С помощью данного параметра можно выбрать либо все направления, либо конкретное направление.
   Диаграмма строится по индикатору, выбранному в таблице.
   На диаграмму влияет параметр с условием: выводить либо сравнение плана с фактом, либо процент исполнения плана.
