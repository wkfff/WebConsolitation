Назначение:отчет предназначен для анализа социально-экономического положения Ямало-Ненецкого автономного округа по основным направлениям развития.
Параметры:выбираются следующие параметры:
• Период;
• Периодичность показателей (год, квартал, месяц).
В отчете представлены данные по основным направлениям развития социально-экономического положения Ямало-Ненецкого автономного округа в зависимости от выбранного периода.
По определенным показателям приводится оценка динамики по отношению к предыдущему периоду: абсолютное отклонение, а также темп роста.
Для показателей отображается визуальная индикация. Рост показателя иллюстрирует стрелка вверх, а снижение показателя – стрелка вниз. Причем, если динамика показателя характеризуется как положительная (увеличение числа родившихся, величина прожиточного минимума и т.п.), то стрелка имеет зеленый цвет; если динамика показателя характеризуется как отрицательная, то появляется стрелка красного цвета.
В отчете построена диаграмма динамики выбранного в таблице (по строкам) показателя.
