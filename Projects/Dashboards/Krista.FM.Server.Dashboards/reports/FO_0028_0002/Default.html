Источник данных:Система бюджетного учета АС "Смета". Данные по предоставленным кредитам с 2006 года.
Назначение отчета:анализ динамики погашения задолженности по бюджетным кредитам, выданным из областного бюджета.
Параметры: год, организация и договор. После изменения параметров нажмите кнопку «Обновить».
В отчете приводятся данные по погашению задолженности по бюджетным кредитам, предоставленным из областного бюджета по следующим показателям:
  • задолженность на начало года, который отражает остаток основного долга на начало года;
  • изменение задолженности за год, который рассчитывается как разница между выданной и погашенной суммой основного долга.
Ежегодно к отчету добавляются данные за соответствующий год.
