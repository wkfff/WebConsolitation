Источник данных:отчетность Федеральной налоговой службы о налоговой базе и структуре начислений по местным налогам, передаваемая в соответствии с Приказом ФНС России от 23 декабря 2005 г. N САЭ-3-10/686@.
   Назначение:Отчет предназначен для анализа начисленных сумм по земельному налогу в разрезе категорий налогоплательщиков (юридических и физических лиц.
   Параметры: Управляющим параметром является «Год». После выбора года необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний период, по которому есть данные в базе.
В таблице отчета приводятся суммы начислений земельного налога по физическим и юридическим лицам, а также оценивается доля начислений по каждой категории налогоплательщиков в общей сумме начислений. Информация приводится в разрезе городских округов, муниципальных районов и поселений. По показателям доли оценивается ранг муниципальных образований. Желтой звездой индицируется максимальная среди муниципальных образований доля начислений, серой звездой – минимальная.
В диаграмме отчета приводится сумма начислений по физическим и юридическим лицам в разрезе муниципальных районов и городских округов. По умолчанию в диаграмме выводятся только муниципальные районы. Для отображения городских округов необходимо отключить параметр «без ГО».
