Источник данных: отчетность об исполнении консолидированного бюджета субъекта РФ, бюджета ТГВФ, областного бюджета и местных бюджетов, база планирования бюджета.

Назначение отчета: отчет предназначен для анализа структуры расходов бюджетов.

Параметры: Выбирается одна из опций:

  • «Фактические данные» - в отчет выводятся данные по фактическому исполнению, согласно месячным отчетам об исполнении бюджетов;
  • «Плановые данные» - в отчет выводятся утвержденные годовые назначения, согласно месячным отчетам об исполнении бюджетов;
  • «Проект бюджета» - в отчет выводятся данные проектов бюджетов по доходам и расходам.

Для опций «Фактические данные» и «Плановые данные» доступны параметры «Год» «Месяц» (по умолчанию установлены последние, по которым имеются данные), «Бюджет» (можно выбрать консолидированный бюджет, бюджет субъекта и местные бюджеты).

Для опции «Проект бюджета» в отчет выводятся параметры:

  • «Год» «Месяц»;
  • «Вариант доходов» - выбирается год, который является текущим, и вариант проекта доходов в указанном году;
  • «Бюджет».

После изменения параметров нажмите кнопку «Обновить».

В таблице по соответствующему уровню бюджета отображаются:

  • Сумма фактического исполнения (бюджетных назначений, проектные данные) по соответствующим расходам.
  • Доля выбранного вида расходов (группы расходов) в общей сумме расходов бюджета.
  • Сумма фактического исполнения (бюджетных назначений, проектные данные) за аналогичный период предыдущего года.
  • Темп роста расходов к аналогичному периоду прошлого года.

По выбранному периоду и уровню бюджета отображается круговая диаграмма с частью значений (социальная сфера), вынесенных в гистограмму.
