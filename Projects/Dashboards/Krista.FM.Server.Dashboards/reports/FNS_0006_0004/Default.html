Назначение отчета: отчет предназначен для анализа поступлений собственных доходов и мониторинга недополученных средств в консолидированный бюджет субъекта. В отчете содержится информация за ряд лет, начиная с 1995 года по текущий период.

Параметры: выбирается перечень лет, после выбора необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний месяц, по которому имеются данные и за десть последних лет. Денежные значения в отчете отображаются в млн.руб.

В первой таблице за ряд лет приводится фактическое исполнение консолидированного бюджета субъекта за каждый месяц и квартал года с расчетом итога за год и удельного веса квартальных поступлений в году.

Во второй таблице за ряд лет приводятся данные о недополученных доходах с разделением общей суммы недоимки на виды налогов по которым были недополучены средства:

  • федеральные налоги
  • региональные налоги
  • местные налоги
  • налоги со спец. налоговый режимом

Диаграмма отчета отображает долю недополученных доходов в общем поступлении собственных доходов за ряд прошлых лет. По умолчанию оценка доли недоимки приведена в «абсолютных показателях», для построения диаграммы по относительным данным необходимо выбрать параметр «относительные показатели».

В отчете реализована возможность выгрузки информации в формат xls и pdf для последующего анализа.

При нажатии сноски «Источники данных» выводится перечень источников, на основе которых построен отчет.
