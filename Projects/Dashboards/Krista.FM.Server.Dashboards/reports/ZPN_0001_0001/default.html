Назначение:отчет предназначен для анализа динамики среднемесячной заработной платы и НДФЛ в разрезе ОКВЭД в субъектах Российской Федерации, входящих в Уральский федеральный округ (по данным органов государственной власти субъектов РФ).
   Параметры:выбирается:
   - территория: субъекты РФ, входящие в Уральский Федеральный округ либо Уральский Федеральный округ в целом;
   - вид деятельности;
   - период (год).
   После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация по Курганской области за 2010 год.
   Для выбранного вида деятельности формируется:
   1. Диаграмма динамики по следующим показателям: «Объем среднемесячной заработной платы», «НДФЛ начислено» и «НДФЛ расчетный (13%)» (рассчитанный исходя из объема среднемесячной заработной платы) по выбранного субъекту;
   2. Гистограмма с накоплением строится по показателю «Задолженность по НДФЛ» по всем субъектам, входящим в УрФО;
   3. Диаграмма Парето строится по показателю «НДФЛ начислено» по всем субъектам, входящим в УрФО.
