Источник данных:
  • ежемесячные отчеты об исполнении бюджетов субъектов РФ в формате html, размещаемые на сайте федерального казначейства РФ (http://www.roskazna.ru/reports/mb.html);
  • данные долговых книг субъектов размещаемые на сайте Министерства финансов РФ (http://www1.minfin.ru/ru/public_debt/subdbt/);
  • данные территориального органа государственной статистики по субъекту РФ о численности постоянного населения на начало года.
 Назначение:отчет предназначен для сравнительного анализа среднедушевых доходов субъектов РФ.
Параметры:выбираются год, месяц (по умолчанию установлены последние, по которым имеются данные) и субъект. После изменения параметров нажмите кнопку «Обновить».
В отчете отображается точечная диаграмма. Каждый субъект представлен точкой. По оси X расположена численность населения. Чем дальше точка вправо, тем более населенный субъект. По оси Y расположены среднедушевые доходы. Чем выше точка, тем больше среднедушевые доходы. Пунктирной линией приведено среднее значение среднедушевых доходов.
Точка выбранного в параметрах субъекта подсвечивается оранжевым цветом.
При наведении курсора мыши на точку выводится подсказка с информацией по субъекту.
Отчет позволяет визуально анализировать распределение субъектов по среднедушевым доходам.
Если установить отметку в поле «Субъекты одного федерального округа», то будет производиться сравнение только в рамках федерального округа, иначе – среди всех субъектов РФ.
Если установить отметку в поле «Не учитывать дотации на выравнивание бюджетной обеспеченности», то из объема доходов будут исключены эти дотации.
