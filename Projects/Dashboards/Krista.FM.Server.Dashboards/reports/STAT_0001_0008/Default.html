Источник данных:данные мониторинга социально-экономическим показателя по структуре инвестиций в основной капитал в субъектах РФ, входящих в Уральский Федеральный округ (по данным органов государственной власти субъектов РФ и федеральной службы государственной статистики).
   Назначение:отчет предназначен для анализа динамики и структуры инвестиций в основной капитал по субъектам РФ, входящих в Уральский Федеральный округ в разрезе:
  • видов деятельности;
  • форм собственности;
  • источников финансирования;
  • видов основных фондов.
   Параметры: выбирается:
  • территория: субъекты РФ, входящие в Уральский Федеральный округ либо Уральский Федеральный округ в целом;
  • разрезность: по видам экономической деятельности, по формам собственности, по источникам финансирования, по видам основных фондов.
   После выбора параметров необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация в целом по Уральскому Федеральному округу в разрезе видов экономической деятельности.
   В отчете по каждому показателю приводится оценка динамики – темп роста к абсолютному показателю по отношению к предыдущему периоду (к предыдущему кварталу).
   Для выбранного субъекта формируется диаграмма динамики по показателю «Структура инвестиций в основной капитал» в разрезе видов деятельности за 2 квартал 2010 года и диаграмма динамики c накоплением по годам по показателю «Структура инвестиций в основной капитал» в разрезе видов деятельности.
