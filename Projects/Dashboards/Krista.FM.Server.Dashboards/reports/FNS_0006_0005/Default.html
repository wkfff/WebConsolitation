Назначение отчета: отчет предназначен для анализа поступлений собственных доходов и мониторинга недополученных средств в консолидированный бюджет муниципального образования. В отчете поводиться сравнение показателей исполнения бюджета анализируемого муниципального образования с средними показателями исполнения по региону.

Параметры: выбирается год, месяц, и муниципальное образование, после выбора необходимо нажать кнопку «Обновить». По умолчанию в отчете выводится информация за последний месяц, по которому имеются данные и первый городской округ по алфавиту. Вывод денежных значений в отчет зависит от параметров «тыс.руб.» и «млн.руб.», по умолчанию данные предоставляются в тыс.руб. При выборе параметров «средние показатели по районам (городам)» и «показатели субъекта» в первой таблице и диаграмме отчета появляется дополнительная информация о средних показателях исполнения бюджета в регионе.

Состав показателей в строках первой таблицы зависит от параметра «Вид дохода», - все выбранные элементы параметра будут отображаться в таблице.

Во второй таблице приводятся данные о недополученных доходах с разделением общей суммы недоимки на виды налогов по которым были недополучены средства:

  • федеральные налоги
  • региональные налоги
  • местные налоги
  • налоги со спец. налоговый режимом

В этой же таблице, для сравнения, представлена информация об общей недоимке по районам(городам) субъекта и в целом по региону.

Диаграмма отчета отображает общую картину исполнения бюджетов муниципальных образований субъекта по нескольким показателям с отображением средних значений показателей в регионе. Показатель, по которому необходимо выводить информацию на диаграмму, выбирается в параметре «Показатель». Вид дохода, на основе которого должен рассчитываться показатель, выбирается в параметре «Доходы».

В отчете реализована возможность выгрузки информации в формат xls и pdf для последующего анализа.

При нажатии сноски «Источники данных» выводится перечень источников, на основе которых построен отчет.
