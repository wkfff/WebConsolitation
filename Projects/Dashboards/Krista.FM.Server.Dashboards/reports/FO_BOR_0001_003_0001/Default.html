Назначение: отчёт предназначен для отражения итогов оценки результативности деятельности ГРБС по трём направлениям.

   1. Результативность деятельности ГРБС по достижению целей.

   2. Общая результативность деятельности ГРБС по решению тактических задач каждой цели.

   3. Результативность деятельности ГРБС по решению каждой тактической задачи.

   Параметры: для построения отчёта выбираются следующие параметры: период анализа и орган исполнительной власти, для которого необходимо вывести результаты оценки. Данные параметры влияют на отображение данных всего отчёта.

   После выбора параметров необходимо нажать кнопку «Обновить».

   В заглавной таблице отражаются данные для выбранного ГРБС:

   - лимит финансирования на указанный период;

   - фактическое исполнение за указанный период;

   - индекс затрат за указанный период;

   - показатель общей результативности достижения целей за указанный период;

   - показатель общей результативности деятельности ГРБС в целом за указанный период (итоговый показатель).

   После выбора показателя в соответствующем параметре, выводится дополнительная таблица с теми же сведениями по выбранному показателю.
